## Usage

### Set Up

[_Make sure you have yarn installed_](https://yarnpkg.com/lang/en/docs/install/)

```bash

git clone https://github.com/mlacademy/frontend.git

cd frontend

cp .env.example .env

yarn install

```

_Change variables in .env_

### Run Locally

```bash

# In frontend/src

yarn start

```
